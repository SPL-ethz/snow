ethz_snow¶

This is the documentation of ethz_snow. A package for the simulation of stochastic nucleation of water in small vials. Please have a look at the Overview for a high-level description of what this package does, or check out the Tutorial: Build your first project.

Getting Started¶

This package can be installed via pip. Simply use

pip install ethz_snow

or see the Tutorial: Build your first project for more details.
