ethz_snow¶

This is the documentation of ethz_snow. A package for the simulation of stochastic nucleation of water in small vials. Please have a look at the Overview for a high-level description of what this package does, or check out the Sphinx tutorial.

Getting Started¶

This package can be installed via pip. Simply use

pip install ethz_snow

or see the Sphinx tutorial for more details.
